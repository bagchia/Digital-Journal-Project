1. Make sure you have a recent version of Node.js installed
2. Make sure you are on the OSU network or connected to the OSU VPN in order for the backend to communicate with the database
3. In your console navigate to the backend folder
4. Run: npm install
5. Run: node app.js
6. Open a new console window (keep the other one open)
7. Navigate to the frontend folder
8. Run: npm install
9. Run: npm start
10. Visit localhost:3000 in your browser
